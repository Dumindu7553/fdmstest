# FDMS - Finance Device Management System

This is a Full-Stack Finance Device Management System. It is designed to manage and monitor financial devices through a mobile app and a central web dashboard.

## Project Components

* *Android-MDM*: The mobile client application used on financial devices for management and monitoring.
* *Backend*: A Node.js and Express.js server that handles device data, security, and API communication.
* *Frontend*: A React/TypeScript web dashboard for administrators to manage devices.

## Technologies Used

* *Frontend*: React, TypeScript
* *Backend*: Node.js, Express.js
* *Mobile*: Android SDK
* *Database*: Local Data Storage / SQLite

## Key Features

* Device Registration and Tracking
* Financial Device Policy Management
* Real-time Data Sync between Mobile and Server
